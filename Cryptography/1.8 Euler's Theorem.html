1.8 Euler's Theorem

Statement of Euler’s Theorem

If n is a positive integer and a is an integer coprime to n (meaning gcd(a, n) = 1), then:
aφ(n) ≡ 1 (mod n) where φ(n) is Euler’s totient function (the number of integers less than or equal to n that are coprime to n).

Explanation

Raising a to the power of φ(n) always gives a result congruent to 1 modulo n, if a and n are coprime.

Fermat’s Little Theorem

If n = p is prime, then φ(p) = p − 1, so Euler’s theorem reduces to:

ap − 1 ≡ 1 (mod p)
