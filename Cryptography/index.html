Cryptography

Cryptography is a fascinating field that relies on mathematical properties of numbers. The main implementations of cryptography involve securing data, and this is done by changing the form in which data is presented so that it could only be accessible to the intended receiver (encrypting data), or creating digital signatures to prove one's identity on the internet. Hashing is another implementation of cryptography where a unique fingerprint is created for a string of data to ensure integrity.
