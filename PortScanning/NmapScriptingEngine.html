The Nmap Scripting Engine

The Nmap Scripting Engine (NSE) is a feature in Nmap that lets you run scripts (written in Lua) to automate advanced network discovery, vulnerability detection, and exploitation tasks.

Finding scripts

nmap scripts are located in the /usr/share/nmap/scripts directory, you can search through them using the grep utility.

$ ls /usr/share/nmap/scripts | grep [name of the service you want to scan]

Finding info about scripts

$ nmap --script-help <script-name>

Updating your script database

$ nmap --script-updatedb

Using the scripts

$ nmap -p [port] --script [script name] [target]
  • Separate multiple scripts by a comma.

If you'd like to use all the scripts related to a certain service, you could use a wildcard:

$ nmap -p 21 --script "*ftp-*" [target]
// runs all ftp scripts on the target
