Host Discovery

Host discovery is the process of identifying active hosts on a target subnet or network. By active we mean that the computer is actually on and connected to the network.

Using nmap

$ nmap -sn [IP]

// -sn for ping scan
// the IP you'd specify here would be the IP of a whole network, like 10.0.2.1-255 or 10.0.2.0/24

Using netdiscover

Netdiscover is a simple network reconnaissance tool used to find live hosts (devices) on a local network. It sends ARP requests on the local subnet, and any active host should reply with its IP address and MAC address.

$ netdiscover -r [range of IP addresses]

// sends ARP requests to the whole range in order to determine which hosts are up
